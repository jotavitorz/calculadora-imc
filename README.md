# 🧮 Calculadora de IMC

Este projeto é uma aplicação web simples que calcula o **IMC (Índice de Massa Corporal)** do usuário com base em seu peso e altura.

O sistema foi desenvolvido utilizando **HTML**, **CSS** e **JavaScript puro**, com foco na prática de manipulação de DOM, eventos e estruturação de páginas responsivas.

## ✅ Funcionalidades

- Entrada de peso e altura
- Cálculo automático do IMC ao enviar o formulário
- Classificação do resultado com base nas faixas de IMC:
  - Abaixo do normal
  - Normal
  - Sobrepeso
  - Obesidade grau I, II e III
- Validação de campos numéricos

## 🧠 Objetivo do projeto

Este projeto tem fins didáticos, sendo parte do meu aprendizado em desenvolvimento web. Serve como prática para:
- Trabalhar com formulários HTML
- Tratar eventos com JavaScript
- Realizar validações simples
- Manipular o DOM para exibir resultados dinâmicos
